📊 End-to-End Data Analytics Portfolio Project
📌 Overview

This project demonstrates a complete end-to-end data analytics workflow, from loading and cleaning data to analyzing it with SQL and creating an interactive Power BI dashboard. The goal was to transform raw data into meaningful business insights using industry-standard tools.

📂 Dataset

The project uses a retail sales dataset containing customer, product, sales, shipping, and review information. The data was cleaned and analyzed to identify trends and support business decisions.

🚀 Project Steps
1. Data Loading
Imported the dataset using Python.
Explored the data structure and data types.
2. Exploratory Data Analysis (EDA)
Checked missing values and duplicates.
Analyzed trends using charts and summary statistics.
3. Data Cleaning
Removed duplicates.
Handled missing values.
Standardized and prepared the data for analysis.
4. SQL Analysis
Imported the cleaned data into PostgreSQL, MySQL, and SQL Server.
Performed SQL queries using joins, CTEs, window functions, aggregations, and business analysis.
5. Power BI Dashboard
Built an interactive dashboard with KPIs, charts, and slicers.
Visualized revenue, customer behavior, product performance, and shipping trends.
6. Report & Presentation
Prepared a business report with key insights.
Created a project presentation using Gamma AI.
