# 📊 Customer Behavior Data Analysis | End-to-End Data Analytics Project

## 📌 Overview

This project demonstrates a complete end-to-end data analytics workflow, transforming raw retail customer data into actionable business insights. The analysis covers every stage of the analytics process—from data cleaning and exploratory analysis to SQL-based business reporting and interactive dashboard creation in Power BI.

The objective of the project was to analyze customer purchasing behavior, product performance, and sales trends to support data-driven business decisions using industry-standard analytics tools.

---

## 🎯 Business Objectives

The project aims to answer key business questions such as:

- Who are the most valuable customer segments?
- Which product categories generate the highest revenue?
- How do customer demographics influence purchasing behavior?
- Which shipping methods perform the best?
- What sales trends can help improve business performance?

---

## 📂 Dataset

The project uses a retail sales dataset containing customer, product, order, shipping, and review information.

### Dataset Includes

- Customer Details
- Product Information
- Order Details
- Sales & Revenue
- Shipping Data
- Customer Reviews
- Discounts
- Purchase Dates

The dataset was cleaned and prepared before analysis to ensure accurate reporting.

# 🔄 Project Workflow

- Performed initial data inspection.

## 2️⃣ Exploratory Data Analysis (EDA)

Performed exploratory analysis to better understand the dataset.

Tasks included:

- Missing value analysis
- Duplicate detection
- Distribution analysis
- Correlation analysis

## 3️⃣ Data Cleaning

Prepared the dataset for analysis by:

- Formatting data types
- Preparing clean analytical datasets

## 4️⃣ SQL Analysis

Imported the cleaned dataset into multiple database systems:
